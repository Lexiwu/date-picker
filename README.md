# date-picker

date-picker by using react

### How to run

1. install dependencies

```
yarn
```

2. start server

```
yarn dev
```

3. Check the site at `http://localhost:3000`

### How to build

```
yarn build
```

if you want to preview and run the code below

```
yarn preview
```

Check the site at `http://localhost:8080/`
